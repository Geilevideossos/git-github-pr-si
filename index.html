🎯 The Concept

✅ Giant 3D-Printed Case with chessboard on top

✅ 2D Gantry moves an Electromagnet in an X & Y Grid

✅ Magnetic Chess Pieces with permanent magnets on bottom

✅ Algorithm-Powered ESP32 plays against user

✅ 4x4 Keypad for user input (A-H + 1-8)

✅ LCD Display shows board state & error messages

⚡ Power Distribution

24V Power Supply
AC/DC Converter Main Power for elektromagnet
↓ 24V → 12V Step down module
Stepper Motors + Driver Boards
↓ 12V → 5V Step down module
ESP32 Microcontroller
📡 ESP32 Controls Everything
Algorithm Chess Engine + Gantry Movement + Display

🤖 2D Gantry System

📐 Two Motion Axes:

  • X-Axis: Left-Right movement (four graveyard and A-H columns)
  • Y-Axis: Forward-Backward movement (1-8 rows)

Movement controlled by stepper motors for ±2mm accuracy

Can reach all 64 squares + 32 graveyard positions

🧲 Electromagnet Mechanism

Power & Force:

  • 24V 6A electromagnet mounted on the gantry
  • STRONG pulling force: ~5kg magnetic power

Piece Interaction:

  • Permanent magnets glued to bottom of each piece
  • Distance: ~3mm below actual piece surface

⚙️ Precision & Positioning

Motion Control:

  • NEMA 17 Stepper Motors on both axes
  • Microstepping for smooth movement
  • Independent X & Y control via ESP32

Accuracy & Coverage:

  • ±2mm positioning accuracy per square
  • Covers standard 8x8 board + side graveyard areas

🎮 User Control System

⌨️ 4x4 Keypad Layout:

A B C D
E F G H
1 2 3 4
5 6 7 8

📍 How to Play:

  • Step 1: Press Source Square (e.g., E2 for pawn)
  • Step 2: Press Target Square (e.g., E4 where to move)
  • Gantry automatically moves piece!

  • Real-time tracking of game state

Updates:

  • After each move (human or algorithm)
